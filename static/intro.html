Topic－Visor：展示你想要的主题！

创建你自己的主题模型的可视化展示系统.

本可视化平台，可以将用户挖掘出的主题信息，通过交互式可视化的形式，直观生动的表达出来。您可以通过选择查看不同的demo来，选择适合自己需求的可视化系统。

数据范例

为提供完整的可视化系统服务，您需要提供4项JSON文件： Article.json, Category.json, Topic-Category,json, Topic.json. 每个JSON文件的范例格式如下：

Article.json

文本的元数据信息，实例请点击下载


{
"article-ID": {
    "abstract" : "strings",
    "authors" : "string",
    "id" : "string",
    "keywords": "string"
    "title" : "string"
},

    ...
}

Topic.json

主题挖掘生成的topic信息，点击下载


{
"id" : {
    "data" : "string",
    "wordlist" : {
        "word" : number,
        ...
    },
    "weight" : number
},
...
}

Category.json

Paper对应信息,点击下载demo


{
    {
        "id": "string"
    },
    {
        "id": "string"
    },
    ...

}
